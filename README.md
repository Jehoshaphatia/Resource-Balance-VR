# VR Resource Balancing Game
![Alt text](/Screenshots/Screenshot.PNG?raw=true "Gameplay")
Resource Balance is a mobile VR game that challenges players to balance the growth of cities against the environmental consequences. Mining natural resources allows the player to build cities and increase wealth, but each city's prosperity also depends on access to clean water and fertile land. The player wins by maximizing the combined prosperity
## Versions
- Unity 2017.2.0f3
- GVR Unity SDK v1.120.0
